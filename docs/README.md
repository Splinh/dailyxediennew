- [Checklist JS theme HD tham khảo](REVIEW-hd-js-tham-khao.md): tài liệu kỹ thuật từ theme HD gốc, chỉ dùng để tham khảo.

## Quy Ước

- Cập nhật tiến độ triển khai thực tế trong `REVIEW-can-chinh.md`.
- Cập nhật thông tin khách hàng và dữ liệu còn thiếu trong `client-info.md`.
